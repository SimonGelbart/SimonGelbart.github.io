# SimonGelbart.github.io

Personal engineering lab site for Simon Gelbart.

This repository hosts a minimal static GitHub Pages site focused on backend tooling, architecture-heavy libraries, deterministic systems, and practical software design.

## Local preview

The site is intentionally static and has no build step.

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Structure

- `index.html` — homepage
- `projects/index.html` — project overview
- `notes/index.html` — design notes index
- `about.html` — short profile
- `assets/styles.css` — shared styling
