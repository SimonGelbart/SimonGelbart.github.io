# Documentation hub

This site is the shared public shell for Simon Gelbart projects, learning paths, and cross-project notes.

Project repositories remain the source of truth for their own documentation. The main site gives those documents a coherent entry point and visual identity.

## Current role

- Home for portfolio-level positioning.
- Index for project documentation and source repositories.
- Index for learning paths.
- Place for global notes and publishing direction.

## Near-term migration

The first goal is to prove the Astro/Starlight shell on the main site. After that, the learning lab can migrate first, followed by Pattrn and LORQ.
